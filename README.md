# Smart Farming API AKSA

## Overview
A minimal FastAPI service that:
- Retrieves open‑source weather data (Open‑Meteo)
- Serves dummy sensor readings
- Accepts an optional field image and generates a description using Ollama **minimax-m3:cloud**
- Provides fertilizer recommendations via Ollama **gpt‑oss:120b-cloud** using a simple in‑memory RAG store.

## Prerequisites
1. **Python 3.9+**
2. **Ollama** installed and running locally (`http://localhost:11434`).
3. Pull the required Ollama models:
   ```bash
   ollama pull gpt-oss:120b-cloud
   ollama pull minimax-m3:cloud
   ollama pull nomic-embed-text
   ```
4. Git (optional, for cloning the repo).

## Setup
```bash
# Clone the repo (if not already)
git clone <repo-url>
cd Project/TA

# Install Python dependencies
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Running the API
```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000
```
The service will be available at `http://localhost:8000`.

## Endpoints
| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/` | Health check – returns `{"message": "Smart Farming API is running"}` |
| `GET` | `/weather?lat={float}&lon={float}` | Fetch current weather from Open‑Meteo. |
| `GET` | `/sensor` | Returns dummy sensor data (soil moisture, nutrients, temperature). |
| `POST` | `/image` | Upload an image (`multipart/form-data`) – returns a description generated by llava. |
| `POST` | `/recommend` | Body: `{"prompt": "..."}` with optional image upload. Returns a concise fertilizer recommendation. |

### Example curl requests
```bash
# Health check
curl http://localhost:8000/

# Weather (example coordinates for Jakarta)
curl "http://localhost:8000/weather?lat=-6.2&lon=106.8"

# Sensor data
curl http://localhost:8000/sensor

# Image analysis (replace path/to/image.jpg)
curl -X POST -F "image=@path/to/image.jpg" http://localhost:8000/image

# Recommendation (text only)
curl -X POST -H "Content-Type: application/json" \
  -d '{"prompt": "What fertilizer should I apply today?"}' \
  http://localhost:8000/recommend

# Recommendation with an image
curl -X POST -F "image=@path/to/image.jpg" -F "prompt=What fertilizer should I apply?" \
  http://localhost:8000/recommend
```
